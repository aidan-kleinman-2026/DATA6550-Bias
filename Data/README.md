## FairFace Dataset

Due to GitHub storage limitations, image files are not included in this repository.

To reproduce our analysis:
1. Download the FairFace dataset from Kaggle:
   https://www.kaggle.com/datasets/jessicali9530/fairface
2. Extract the dataset into the `Data/` directory
3. Ensure the following structure:
   Data/
    ├── train/
    ├── val/
    ├── fairface_label_train.csv
    └── fairface_label_val.csv
